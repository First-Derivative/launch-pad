## 1. Git Setup

### Initialize Repository

```sh
cd ~/dev/launch-pad

# Initialize git
git init

# Create .gitignore
cat << 'EOF' > .gitignore
# Binaries
launch
launch-pad
*.exe
dist/

# IDE
.idea/
.vscode/
*.swp

# OS
.DS_Store
Thumbs.db
EOF

# Initial commit
git add .
git commit -m "Initial commit: launch-pad CLI"
```

### Create GitHub Repository

```sh
# Create repo on GitHub (requires gh CLI)
gh repo create ashhatz/launch-pad --public --source=. --remote=origin

# Push
git push -u origin main
```

**Manual alternative:** Create at https://github.com/new with name `launch-pad`

## 2. GoReleaser Setup

[GoReleaser](https://goreleaser.com/) automates cross-compilation and GitHub releases.

### Install GoReleaser

```sh
brew install goreleaser
```

### Create Configuration

Create `.goreleaser.yaml` in the project root:

```yaml
# .goreleaser.yaml
version: 2

project_name: launch-pad

before:
  hooks:
    - go mod tidy

builds:
  - id: launch-pad
    binary: launch
    main: ./main.go
    env:
      - CGO_ENABLED=0
    goos:
      - darwin
      - linux
    goarch:
      - amd64
      - arm64
    ldflags:
      - -s -w
      - -X main.version={{.Version}}

archives:
  - id: default
    format: tar.gz
    name_template: "{{ .ProjectName }}_{{ .Os }}_{{ .Arch }}"

checksum:
  name_template: "checksums.txt"

changelog:
  sort: asc
  filters:
    exclude:
      - "^docs:"
      - "^test:"
      - "^chore:"

brews:
  - name: launch-pad
    repository:
      owner: ashhatz
      name: homebrew-launch-pad
      token: "{{ .Env.HOMEBREW_TAP_TOKEN }}"
    directory: Formula
    homepage: "https://github.com/ashhatz/launch-pad"
    description: "A tmux session launcher"
    license: "MIT"
    install: |
      bin.install "launch"
    test: |
      system "#{bin}/launch", "--help"
```

### Test Locally

```sh
# Dry run (no publish)
goreleaser release --snapshot --clean

# Check dist/ folder for built binaries
ls -la dist/
```

## 3. Homebrew Tap Setup

### Create Tap Repository

```sh
# Create the homebrew tap repo
gh repo create ashhatz/homebrew-launch-pad --public --description "Homebrew tap for launch-pad"

# Clone it locally (optional, GoReleaser will manage it)
git clone https://github.com/ashhatz/homebrew-launch-pad.git ~/dev/homebrew-launch-pad
```

### Repository Structure (auto-generated by GoReleaser)

```
homebrew-launch-pad/
└── Formula/
    └── launch-pad.rb    # Auto-generated on release
```

### Usage After Setup

Users will install via:

```sh
brew tap ashhatz/launch-pad
brew install launch-pad
```

Or in one command:

```sh
brew install ashhatz/launch-pad/launch-pad
```

## 4. GitHub Actions CI/CD

### Create Workflow Directory

```sh
mkdir -p .github/workflows
```

### CI Workflow (`.github/workflows/ci.yaml`)

Runs tests and builds on every push/PR:

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Set up Go
        uses: actions/setup-go@v5
        with:
          go-version: "1.21"

      - name: Build
        run: go build -v ./...

      - name: Test
        run: go test -v ./...
```

### Release Workflow (`.github/workflows/release.yaml`)

Triggers on version tags:

```yaml
name: Release

on:
  push:
    tags:
      - "v*"

permissions:
  contents: write

jobs:
  release:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Set up Go
        uses: actions/setup-go@v5
        with:
          go-version: "1.21"

      - name: Run GoReleaser
        uses: goreleaser/goreleaser-action@v6
        with:
          version: latest
          args: release --clean
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          HOMEBREW_TAP_TOKEN: ${{ secrets.HOMEBREW_TAP_TOKEN }}
```

### Create GitHub Personal Access Token

GoReleaser needs a token to push to the Homebrew tap repo:

1. Go to https://github.com/settings/tokens
2. Generate new token (classic) with `repo` scope
3. Add as secret in `launch-pad` repo:
   - Go to Settings > Secrets and variables > Actions
   - New repository secret: `HOMEBREW_TAP_TOKEN` = your token

## 5. First Release

### Version Your Code (Optional)

Add version support to `main.go`:

```go
package main

import (
	"os"

	"github.com/ashhatz/launch-pad/cmd"
)

var version = "dev"

func main() {
	cmd.SetVersion(version)
	if err := cmd.Execute(); err != nil {
		os.Exit(1)
	}
}
```

And update `cmd/root.go` to expose version:

```go
func SetVersion(v string) {
	rootCmd.Version = v
}
```

### Create and Push Tag

```sh
# Commit any remaining changes
git add .
git commit -m "Add GoReleaser and CI/CD configuration"

# Create version tag
git tag -a v0.1.0 -m "Initial release"

# Push with tags
git push origin main --tags
```

### Verify Release

1. Check GitHub Actions: https://github.com/ashhatz/launch-pad/actions
2. Check Releases: https://github.com/ashhatz/launch-pad/releases
3. Check Homebrew tap was updated: https://github.com/ashhatz/homebrew-launch-pad

## Quick Reference

### Local Development

```sh
# Build
go build -o launch .

# Run
./launch --help
./launch ~/dev/myproject
./launch -a ~/dev/myproject
```

### Cross-Compile Manually

```sh
# macOS ARM (Apple Silicon)
GOOS=darwin GOARCH=arm64 go build -o launch-darwin-arm64 .

# macOS Intel
GOOS=darwin GOARCH=amd64 go build -o launch-darwin-amd64 .

# Linux ARM
GOOS=linux GOARCH=arm64 go build -o launch-linux-arm64 .

# Linux x86_64
GOOS=linux GOARCH=amd64 go build -o launch-linux-amd64 .
```
